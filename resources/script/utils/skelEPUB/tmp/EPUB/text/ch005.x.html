4. Cargar la consulta de Excel

Para cargar consultas en Excel utilizando Power Query, se deben seguir los siguientes pasos:

  • Abre Microsoft Excel y se selecciona la pestaña "Datos" en la cinta de opciones. Se hace clic en "Obtener datos externos" y se selecciona "Desde otras fuentes".

  • En el menú desplegable, se elige "Desde Power Query" y luego se selecciona "Desde el Editor de consultas".

  • Se abrirá la ventana del Editor de consultas. Aquí se puede crear o editar tus consultas de Power Query.

Pestaña datos

  • Si ya se tiene una consulta creada, se puede seleccionarla en el panel de consultas a la izquierda. Si no se tiene ninguna consulta, se puede crear una nueva haciendo clic en "Nueva consulta" y seleccionando la fuente de datos deseada (como un archivo CSV, una base de datos, una página web, etc.).

fuente datos

  • Una vez que se haya seleccionado o creado la consulta, se puede realizar las transformaciones necesarias en los datos utilizando las opciones disponibles en el Editor de consultas.

  • Cuando se haya terminado de realizar las transformaciones, se hace clic en "Cerrar y cargar" en la parte superior del Editor de consultas.

  • Se abrirá una ventana de diálogo donde se puede elegir dónde cargar los datos. Se puede elegir cargar los datos en una nueva hoja de cálculo o en una tabla existente.

  • Se hace clic en "Cargar" y Excel cargará los datos de la consulta en la ubicación seleccionada.

Una vez que se haya cargado una consulta, se puede actualizar en cualquier momento haciendo clic derecho en la tabla cargada y seleccionando "Actualizar" o utilizando la opción "Actualizar todo" en la pestaña "Datos".
