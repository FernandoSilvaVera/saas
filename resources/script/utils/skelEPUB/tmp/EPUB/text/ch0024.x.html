Recuperar una versión anterior de un archivo

Para recuperar una versión de un archivo de los que tenemos en uno de nuestros canales de Teams es necesario abrir el archivo en SharePoint ya que es la única opción que tenemos para ver las opciones de versiones del archivo.

Seleccionamos el archivo y pulsamos abrir en SharePoint, una vez que lo hemos abierto si pulsamos en el botón de más opciones del archivo podemos elegir la opción de historial de versiones que nos permite recuperar versiones anteriores del equipo.

El historial de versiones nos muestra todas las versiones del archivo disponibles, para cada versión tenemos diferentes opciones, ver nos permite ver el archivo para poder comprobar si es el que necesitamos, restaurar nos permite convertir esa versión del archivo en la actual, creándose una versión nueva del archivo anterior a la que restauramos.

También tenemos la opción de eliminar si la versión del archivo no es la última.

Al restaurar el archivo aparece una nueva versión con el archivo restaurado, por lo que si nos hemos equivocado no habría problema porque podemos volver a la versión anterior.
