Conectar diferentes fuentes de datos

Hay que dirigirse a la pestaña "Datos" y seleccionar "Nueva consulta". Luego, ir a la pestaña "Otras fuentes" y elegir "Desde una web" para iniciar una búsqueda.

Fuentes

En el lado derecho de la pantalla, se mostrará un cuadro donde se debe especificar la URL de la página web desde la cual se desea descargar los datos. En este caso, se quiere obtener los datos del S&P 500, los cuales se descargan desde la siguiente URL:

https://en.wikipedia.org/wiki/List_of_S&P_500_companies

Como se ve aparecen varios resultados, el orden puede variar pero solo se necesita "S&P 500 componet stocks[edit]". Es un listado de componentes de S&P 500:

Listado

Se muestran los datos que se van a cargar, por si se desea hacer algún tipo de filtro.

Filtro

A continuación, se continúa explicando cómo agregar los datos:

Para agregar los datos a la hoja de cálculo y comenzar a trabajar con ellos, simplemente haz clic en "Cerrar y cargar".

Power Query ofrece la capacidad de combinar datos de diferentes fuentes. En este caso, vamos a combinar los datos que obtuvimos previamente con los libros de ejemplo relacionados con NYSE y NASDAQ. Puedes descargar estos archivos para realizar ejercicios prácticos haciendo clic en "Libros de ejemplo".

Para cargar estos libros, Power Query proporciona varias opciones para importar datos, como se muestra en la imagen. En esta situación, utilizaremos la opción "Importar desde carpeta".

Ahora, selecciona "Datos" en la barra de herramientas, luego elige "Nueva consulta" y después "Desde un archivo", seguido de "Desde una carpeta". Luego, selecciona la carpeta que contiene los archivos que descargaste.

Importar

Se indica la ruta de la carpeta donde se tienen los datos de ejemplo.

Carpeta datos

Al cargar, aparecerá una ventana en la que muestra todos los datos que contiene esa carpeta, y cierta información sobre ellos.

Información

Si se hace clic en Transformar datos, se puede editar la consulta antes de cargar los datos.
