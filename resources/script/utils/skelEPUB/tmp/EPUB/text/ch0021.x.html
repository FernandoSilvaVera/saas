Generar un cambio en un archivo

Para generar un cambio en un archivo solo tenemos que generar el cambio y si el sistema de control de cambios esta activado se verá reflejado.

  • Si añadimos texto aparece en color rojo subrayado.

  • Si eliminamos texto aparece en color rojo tachado.

Se reflejarán los cambios según tengamos habilitadas las opciones de control de cambios.
